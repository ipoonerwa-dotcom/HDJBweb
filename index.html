// SECURITY

安全设计

借呗的安全不是口号，是代码、审计和治理结构三位一体。

🛡️

双层价格预言机

FLAP Portal Oracle 为主，LP TWAP 为辅。彻底移除可被单笔闪电贷操纵的现货价降级路径。

🔒

Guardian 紧急权

emergencyWithdraw 仅 FLAP 官方多签可调用。Creator 无权提走金库 BNB，从源头堵 rug。

⚖️

硬地板保护

hardFloorBnb + totalOutstanding 双追踪，金库余额永远不会被借款掏空到 0。

🚫

抗灌水队列

MAX_PENDING=50 + 最小借款门槛 + 滑动游标清扫 + 1% keeper 奖励，令占位攻击成本远高于收益。

🔄

MEV 同块防护

S_total 跨块快照，同块操纵借贷的三明治攻击自动失效。

📜

链上透明

合约开源、参数可审计、事件完整。零后门、零管理员提款。
